# **App Name**: OfficeZenith

## Core Features:

- Employee Dashboard: Personalized dashboard for each employee displaying their name, ID, phone number, email, leave balance, and attendance record.
- Admin Dashboard: Comprehensive dashboard for administrators with an overview of total employees, employee attendance, and employee leave requests, all editable.
- Leave Management: System for employees to apply for leave and for admins to approve, reject, or mark leave requests as pending.
- Task Assignment: Functionality for administrators to assign tasks to employees.
- Real-time Notifications: Notification system using a bell icon that blinks for both employees and admins when leave requests are approved, tasks are assigned, or other relevant actions occur. Generative AI tool to learn and determine notification priority.
- Attendance Tracking: System to accurately log and track employee attendance.
- User Authentication: Secure login system for both employees and administrators.

## Style Guidelines:

- Primary color: Deep royal blue (#4169E1) to convey trust and professionalism.
- Background color: Light grey (#F0F0F0) to provide a clean and modern backdrop.
- Accent color: Gold (#FFD700) for highlights and CTAs, lending a luxurious feel.
- Body and headline font: 'Inter' sans-serif, offering a modern and neutral aesthetic for both headlines and body text.
- Use refined, minimalist icons to represent various functions and data points.
- Design a spacious and well-organized layout, leveraging whitespace to achieve a sense of luxury and clarity.
- Employ subtle animations for transitions and feedback, enhancing the user experience without overwhelming the interface.
